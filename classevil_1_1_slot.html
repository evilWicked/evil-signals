Detailed Description

template<typename... Args>
class evil::Slot< Args >

The single threaded specialisation of the SlotBase class.

This class is intended to be used in conjunction with the evil::Signal class. Creates a slot that you attach callbacks to that with accept the arguments defined in the template instantiation.

template<typename... Args>
void evil::Slot< Args >::triggerSlot ( Args...  args)
inline

Triggers the slot passing in the arguments defined in the template. Even though not denoted as such this is a raw function that expects lock guards to be wrapped around it by the caller. It is not normally called by user code but is public so that it can be accessed by the slot.
