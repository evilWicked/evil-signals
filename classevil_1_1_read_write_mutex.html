Detailed Description

The internals of the read write lock - not really a mutex although used like one.

The ReadWriteMutex is NOT a mutex!!! In fact it provides the internals of a read write lock. But then again our ReadWriteLock is not a lock its more of a wrapper around this class. None the less this class - our so called ReadWriteMutex wraps up a mutex with condition variables to provide a parallel reas single write mechanism
