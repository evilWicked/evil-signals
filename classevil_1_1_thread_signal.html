Detailed Description

template<typename... Args>
class evil::ThreadSignal< Args >

The thread safe specialisation of the underlying SignalBase class.

The ThreadSignal class is intended to be used in conjunction with the ThreadSlot class. This version provides a threadsafe specialisation of the underlying signals and slots mechanism. What this guarantees is that you can interact with both the signal and slot from mutilple threads simultaneously without conflict.
