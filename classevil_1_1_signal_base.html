Detailed Description

the base class used by both the evil::Signal and evil::ThreadSignal class.

Have drawn upon the basic outline of a signal defined in http://simmesimme.github.io/tutorials/2015/09/20/signal-slot/ As well as some info on function pointer usage http://www.cplusplus.com/forum/general/159730/
