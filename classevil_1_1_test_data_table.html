Detailed Description

template<typename T>
class evil::TestDataTable< T >

A data table used to capture test results in memory.

A data table to capture the results of testing in memory so that delays due to simultaneous access of the terminal don't impact on results. In simle terms we collect all results and dump at the end rather than while testing is proceeding.
